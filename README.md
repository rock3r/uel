# The Unenshittifiable License (UEL)

The website and canonical text for **UEL v1.0** — a source-available license for software
that belongs to the commons.

> Use it, fork it, learn from it, self-host it, improve it — just don't turn the commons into a toll booth.

🌐 **Live site:** https://uelicense.eu

## What's here

This is a small, dependency-free static site.

| File | Purpose |
| --- | --- |
| [`LICENSE`](LICENSE) | The canonical UEL v1.0 text — copy this into your own project |
| `index.html` | Home — what the license is and who it's for |
| `license.html` | The full license, plain-language first |
| `compare.html` | How UEL compares to MIT, GPL, and AGPL |
| `styles.css` / `app.js` | Shared styling and behaviour |
| `assets/` | Illustrations of the warden goose and the (defeated) toll booth |

## Using the license in your project

Drop the [`LICENSE`](LICENSE) file into your repository, or copy it from the
["Copy license text" button](https://uelicense.eu/license.html) on the site.

## Running locally

It's just static files — open `index.html` directly, or serve the folder:

```sh
python3 -m http.server 8000
# then visit http://localhost:8000
```
